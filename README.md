# Django_project

A project to understand basic django file hierarchy.

## Installation

Use [pip](https://pip.pypa.io/en/stable/)|[conda](https://docs.conda.io/en/latest/) to install django.

```bash
pip install django
```
```bash
conda install -c anaconda django
```
